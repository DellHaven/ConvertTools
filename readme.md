# 数据转换工具
## Version 0.0.2 （Master）
### 功能介绍
#### 支持功能介绍
1. json转csv。
2. 高性能模式。
3. 严谨转换模式。
4. 自动创建结果文件夹及其文件。
5. 正常性能模式的严谨或者不严谨模式下支持显示出异常无法转换成CSV的JSON行Index。
6. 只需要安装Python 3.6.X，无需安装任何第三方模组即可使用。
7. 跨计算机平台。
8. 输入与输出路径自动转换为当前系统可兼容的斜线(正斜线和反斜线)。
#### 不支持的功能
1. json转csv以外的转换模式。
2. 多层嵌套的json。
#### 不推荐的使用
1. JSON的Key或者Value中存在英文的逗号，这会导致CSV列分隔符产生歧义
